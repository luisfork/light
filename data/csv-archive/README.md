# CSV Plan Data Archive

Daily CSV exports for spreadsheet analysis and data science workflows.

## File Naming

`plans_YYYY-MM-DD.csv`

## Purpose

- Excel/Google Sheets analysis
- pandas/R data science workflows
- Database imports (PostgreSQL, MySQL)
- Quick data exploration

## Columns

See `scripts/archive_to_csv.py` for complete column list.

Includes: plan identifiers, pricing tiers, fees, renewable percentage, URLs, and more.
